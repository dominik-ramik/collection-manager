# Collection Manager

A web application for managing biological specimen collections with integrated photo tagging and taxonomy workflows.

## Overview

Collection Manager is a comprehensive tool designed to help biologists and researchers manage field notes, specimen metadata, images, and Symbiota collections data. The application provides an intuitive modular interface for organizing and preparing biological collection data for institutional databases and online portals. Key workflows include:

- **Field Notes Management**: Import and work with spreadsheet field collection data including taxonomy, collector information, and specimen numbers.
- **Specimen Photo Selection**: Browse specimen photo folders, tag images for selection, create edited copies, and navigate between specimens efficiently.
- **Taxa Photo Selection**: Aggregate specimen photos by taxonomic classification, enabling taxon-level photo selection and export.
- **Symbiota Integration**: Prepare specimen data and images for upload to Symbiota-based collections portals.
- **Checklist Management**: Maintain and reference taxonomic checklists with species, subspecies, and family classifications.
